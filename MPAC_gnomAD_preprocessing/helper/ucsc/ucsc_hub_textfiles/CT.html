MPAC CT

This UCSC Genome Browser track displays predicted variant effects in cell-line CT for all variants in gnomAD v3.1.2 on genome build hg38. We predict alternate and reference allele MPRA (Massively Parallel Reporter Assay) activity and compute the allelic skew between them. We used a machine-learning model called MPAC (Malinois with Parallel Aggregated Cross-validation) to do this. This is part of Identifying non-coding variant effects at scale via machine learning models of cis-regulatory reporter assays. Read the paper for more details.

Relevant code can be found on our github. For code specifically used to generate these tracks, consult `MPAC_gnomAD_preprocessing/helper/ucsc` within the repository.
